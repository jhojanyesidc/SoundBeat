# Historias de Usuario – SoundBeat

## HU01 - Ver catálogo de instrumentos
**Como** cliente  
**quiero** ver los instrumentos disponibles  
**para** seleccionar un producto.

## HU02 - Agregar instrumentos al carrito
**Como** cliente  
**quiero** agregar productos al carrito  
**para** realizar la compra.

## HU03 - Finalizar compra y pagar
**Como** cliente  
**quiero** completar el proceso de pago  
**para** recibir mi pedido.

## HU04 - Registrar estudiantes
**Como** administrador  
**quiero** registrar estudiantes  
**para** matricularlos en cursos.

## HU05 - Gestionar cursos
**Como** docente  
**quiero** crear y administrar mis cursos  
**para** impartir clases adecuadamente.

## HU06 - Registrar pagos académicos
**Como** administrador  
**quiero** registrar mensualidades  
**para** llevar control financiero.

## HU07 - Iniciar sesión
**Como** usuario  
**quiero** iniciar sesión según mi rol  
**para** acceder a mi panel correspondiente.
