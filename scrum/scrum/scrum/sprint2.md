# Sprint 2 – SoundBeat
**Duración:** 2 semanas  
**Objetivo:** Implementar funcionalidades de compra y administración de productos.

## Historias de Usuario incluidas
- HU02: Agregar al carrito
- HU03: Finalizar compra y pagar
- HU05: CRUD de productos (administrador)

## Tareas
### Carrito de Compras
- Crear carrito persistente
- Agregar y eliminar productos
- Calcular totales

### Checkout y Pagos
- Crear formulario de pago
- Generar pedido
- Confirmación y resumen

### CRUD Productos
- Crear, editar y eliminar productos
- Gestión de inventario
- Subida de imágenes

## Criterio de Aceptación
- Carrito funciona sin errores
- Checkout genera orden correctamente
- CRUD permite administrar el catálogo
