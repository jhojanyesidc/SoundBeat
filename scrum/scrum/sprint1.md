# Sprint 1 – SoundBeat
**Duración:** 2 semanas  
**Objetivo:** Implementar funcionalidades base para habilitar acceso y visualización inicial.

## Historias de Usuario incluidas
- HU07: Inicio de sesión y roles
- HU01: Ver catálogo de instrumentos
- HU04: Registrar estudiantes

## Tareas
### Autenticación
- Crear formulario de login
- Implementar roles (admin, docente, estudiante, cliente)
- Configurar base de datos de usuarios

### Catálogo
- Mostrar lista de instrumentos
- Filtrar y ver detalle

### Registro de Estudiantes
- Formulario de registro
- Guardar datos en base de datos

## Criterio de Aceptación
- Login funciona según rol
- Catálogo visible al público
- Registro de estudiantes almacena correctamente
